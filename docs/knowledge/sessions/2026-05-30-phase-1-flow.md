# 2026-05-30 — Phase 1: testable flow

Built Phase 1 end to end in 3 reviewed slices on branch `phase-1-flow`. The app went from token-only stub screens to a complete, demoable land → sort → build → results flow with real recommendations. The scoring/data/state "brain" from Phase 0 was reused unchanged; this was screen + interaction work.

## What we did

**Slice 1 — Sort.** Replaced the Phase 0 button-grid with one card at a time into two archetype-neutral bins ("That's me" / "Not my thing") via Motion swipe-drag + tap. Added `RoundIndicator`, `ProgressBar`, a between-round `RoundBeat`, and round-transition copy in new `src/data/rounds.ts` (theme-free per PRD §5.2). Mid-slice user feedback fixed: (a) cards stopped advancing after a drag — an `AnimatePresence mode="wait"` + drag stall (→ `mode="popLayout"` + `forwardRef` + no `x` in exit; see LESSONS L-004); (b) removed the arrow-key/Enter sort mechanic (→ `DECISIONS.md` D-015).

**Slice 2 — Results (the conversion screen).** Three role cards (active centered/full-sized, two ghosted), the four-part read (`FourPartRead`: how-you-match band copy, skills via new `role.skillIds`, competencies `plainName`, top-3 `ProgramList`), `MatchIndicator` (% as H2 in archetype accent), `Pedestal` + `RobotPlaceholder` (placeholder robot tinted by the active archetype via `currentColor`). Compare interaction: tap a card (`tryOnRole`) or drag the robot onto one (hit-test card refs), with a Motion `layout` reflow that re-centers the active card and recolors the robot/pedestal. Low-signal framing via `lib/fit.ts` `isLowSignal`; understated retake. New: `resultsCopy.ts`, `lib/fit.ts` (+ test), `components/accent.ts` (static archetype→class map; LESSONS L-005).

**Slice 3 — Landing + Build.** `src/lib/gsap.ts` registers `useGSAP` + `DrawSVGPlugin` once at app start (side-effect import in `main.tsx`). `LandingSceneHint.tsx` is line-art (belt, parts, arm) that draws in via GSAP DrawSVG inside `useGSAP` + scope, gated by `gsap.matchMedia` for reduced motion; Motion owns the content/CTA entrance (disjoint nodes — no crossed engines; LESSONS L-006). `Build.tsx` is a ~1.4s Motion robot-reveal beat with a nav guard (redirects home if no `scoreResult`). Cleared the React Router v7 future-flag warning (`RouterProvider future={{ v7_startTransition: true }}`).

**Docs.** Captured the refined Phase 2 conveyor model (user drags parts off the belt; arm assembles from the keep bin) across PRD §5.2, DESIGN_SYSTEM §10.3, ARCHITECTURE §5, ROADMAP §3.2/§3.3 + `DECISIONS.md` D-014. D-015 logs the drag+tap (no arrow/enter) call.
